Collected Data (first-party)

To prevent abuse and keep the website functioning, our server may automatically collect the following data:

  • Web browser and/or device
  • Operating system
  • IP Address
  • The time of the activity

Cloudflare (third-party)

Cloudflare is a Content Distribution Network (CDN) used to ensure the website is served quickly and reliably worldwide. It also provides security measures to identify and protect against malicious activity.

To do this Cloudflare may log your IP address and analyse user behaviour while on our site.

Cloudflare may use a cookie named cf_clearance. It's used to detect automated behaviour indicating malicious bots.

You can view Cloudflare's Privacy Policy here.

Cloudflare is used due to our legitimate interest in keeping the website as reliable and secure as possible.
